# Tetris — simple HTML + JS

Это минимальный проект Тетриса на чистом HTML, CSS и JavaScript.

Файлы:
- `index.html` — основная страница
- `style.css` — стили
- `tetris.js` — логика игры

Как запустить:
1. Откройте `index.html` в браузере (двойной клик). Для некоторых браузеров рекомендуется запуск через локальный сервер.

Пример запуска локального сервера с Python (в терминале, в папке проекта):
```bash
cd /Users/oleg/Documents/testProject
python3 -m http.server 8000
# затем открыть http://localhost:8000 в браузере
```

Управление:
- Стрелки ← → — перемещение
- Стрелка ↑ — поворот
- Стрелка ↓ — опускание
- Пробел — быстрый сброс
- P — пауза

Если хотите дополнительные функции (следующий блок, звук, мобильное управление), скажите — добавлю.
